# python
This repository contains a Jupyter Notebook titled task1.ipynb, which includes basic Python programming exercises designed to strengthen fundamental concepts. The notebook covers important topics such as variables and data types, arithmetic operations, number formatting, conditional statements, functions, loops, and the use of built-in modules like random. It includes practical programs such as a simple interest calculator, pond area and water calculation, speed calculation, BMI calculator with classification, city-to-country identification, and a dice roll simulation. The purpose of this project is to build a strong foundation in Python by practicing problem-solving and understanding core programming concepts. This notebook is suitable for beginners who are starting their journey in Python and want hands-on experience with basic programming logic.
